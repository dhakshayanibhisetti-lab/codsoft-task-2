# Email Template UI/UX Design

This project is a modern and responsive email template created using Figma as part of the CodSoft UI/UX Internship Task 2.

## Project Overview

The email template is designed for a food and dessert promotion campaign.  
It includes visually appealing images, promotional sections, CTA buttons, customer reviews, and social media icons.

The design focuses on:
- Clean UI design
- Mobile-friendly layout
- Attractive typography
- Proper spacing and alignment
- User-friendly experience

## Features

- Responsive mobile email layout
- Promotional hero section
- Product showcase images
- Call-to-action buttons
- Customer review section
- Social media integration
- Modern and aesthetic color palette

## Tools Used

- Figma
- PNG Images
- UI/UX Design Principles

## What I Learned

During this project, I learned:
- Basics of UI/UX design
- Creating layouts in Figma
- Working with typography and colors
- Image placement and alignment
- Designing responsive email templates
- Creating clean and user-friendly interfaces
